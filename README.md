# QSkill-Task-1-Student-Data-Analysis
Student Performance Data Analysis using Python, Pandas, Matplotlib and Seaborn.

## 📌 Project Description
This project was developed as part of the QSkill Python Development Internship.

The program reads a CSV dataset of student marks, performs data analysis using Pandas, and visualizes the results using Matplotlib and Seaborn.

## 🚀 Features
- Load CSV dataset
- Display student records
- Show dataset information
- Calculate average marks
- Find highest and lowest marks
- Generate Bar Chart
- Generate Scatter Plot
- Generate Heatmap

## 🛠 Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
